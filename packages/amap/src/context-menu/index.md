## API

### 动态属性

| 属性 |说明|类型|默认值|
|-----|----|----|----|
|content| 右键菜单内容 | `string` \| `HTMLElement` | -- |

### 扩展属性

| 属性 |说明|类型|默认值|
|-----|----|----|----|
|events| 绑定的事件对象 | `object` | -- |


**Item**

| 属性 |说明|类型|默认值|
|-----|----|----|----|
|text| 菜单项的文案 | `string` | -- |
|onClick| 点击的回调 | `() => void` | -- |
